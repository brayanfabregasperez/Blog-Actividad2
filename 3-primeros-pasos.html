Actividad 2: Tutorial sobre Angular.

3. Primeros pasos

En esta sección veremos cómo iniciar un proyecto básico con Angular, tanto usando CDN como con Angular CLI.

Angular es un framework moderno basado en componentes, TypeScript y un sistema de módulos. A continuación tienes los pasos para crear una aplicación mínima.

Hola Mundo - enfoque CLI (Angular)

Angular se utiliza principalmente con Angular CLI. Aquí tienes los pasos mínimos para crear un proyecto y levantarlo en local:

npm install -g @angular/cli
  ng new hola-angular
  cd hola-angular
  ng serve
  # Abrir http://localhost:4200

Un proyecto Angular contiene archivos como src/main.ts, src/app/app.module.ts y componentes en src/app/. A continuación tienes un ejemplo mínimo del componente principal.

Ejemplo básico de componente (Angular)

Ejemplo del componente raíz en Angular (archivo: src/app/app.component.ts y su template):

<!-- Contenedor en el body -->
<!-- src/app/app.component.ts -->
export class AppComponent {
  title = 'Hola desde Angular';
  count = 0;
  increment() { this.count++; }
}

<!-- src/app/app.component.html -->
<div>
  <h3>{{ title }}</h3>
  <p>Contador: {{ count }}</p>
  <button (click)="increment()">Aumentar</button>
</div>

Este componente muestra un título, un contador y un botón para incrementarlo. Angular usa data binding ({{ }}) y event binding ((click)) para enlazar datos y eventos.

El archivo app.module.ts declara el componente y lo arranca:

Recuerda que tienes modificar el archivo app.module.ts para declarar el componente:

import { NgModule } from '@angular/core';
import { BrowserModule } from '@angular/platform-browser';
import { AppComponent } from './app.component';
@NgModule({
  declarations: [AppComponent],
  imports: [BrowserModule],
  bootstrap: [AppComponent]
})
export class AppModule { }

Con estos archivos mínimos, al ejecutar ng serve y abrir http://localhost:4200, verás la aplicación funcionando.
