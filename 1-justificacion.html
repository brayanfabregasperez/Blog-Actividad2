Actividad 2: Tutorial sobre Angular.

1. Justificación del framework (Angular)

Angular es un framework para construir aplicaciones web completas usando TypeScript. Ofrece una solución completa (componentes, enrutamiento, servicios y herramientas para testing y despliegue) ideal para aplicaciones de mediano a gran tamaño.

Creado y mantenido por Google, Angular proporciona una arquitectura estructurada y escalable y un ecosistema con herramientas oficiales (Angular CLI, Angular Material, Angular Universal) que facilitan el desarrollo profesional.

a) ¿Para qué sirve?

Angular sirve para desarrollar aplicaciones web dinámicas y de una sola página (SPA) con una experiencia de usuario fluida. Permite crear interfaces interactivas y responsivas mediante el uso de componentes reutilizables, gestión del estado y comunicación eficiente con APIs backend.

Además, Angular facilita la organización del código, la modularidad y el mantenimiento de aplicaciones complejas, gracias a su arquitectura basada en componentes y servicios. También incluye características avanzadas como inyección de dependencias, enrutamiento, formularios reactivos y herramientas para pruebas automatizadas.

Estas capacidades hacen que Angular sea una opción popular para desarrollar aplicaciones empresariales, paneles de administración, plataformas de comercio electrónico y otras aplicaciones web que requieren una experiencia de usuario rica y un rendimiento óptimo.

Angular también es adecuado para proyectos que requieren escalabilidad y mantenibilidad a largo plazo, ya que su estructura y herramientas integradas facilitan la colaboración entre equipos de desarrollo y la evolución continua del software.

b) ¿Qué aplicaciones tiene?

Angular se utiliza en aplicaciones web empresariales, paneles de administración, plataformas de comercio electrónico, aplicaciones de gestión de contenido (CMS), aplicaciones de redes sociales y aplicaciones móviles híbridas mediante Ionic.

Empresas como Google, Microsoft, IBM, PayPal y Upwork utilizan Angular para desarrollar aplicaciones web complejas y escalables.

Su capacidad para crear interfaces de usuario interactivas y gestionar grandes volúmenes de datos en tiempo real lo hace adecuado para aplicaciones financieras, educativas y de salud, entre otras. Además, su ecosistema robusto y su arquitectura modular facilitan la integración con otras tecnologías y la adaptación a diferentes necesidades empresariales.

Angular es especialmente útil en proyectos donde se requiere una estructura sólida y mantenible, así como en entornos de desarrollo colaborativos donde múltiples desarrolladores trabajan en el mismo código base.

c) ¿Sobre qué lenguajes se apoya?

Angular se apoya principalmente en TypeScript, un superconjunto de JavaScript que añade tipado estático y características avanzadas. TypeScript mejora la productividad del desarrollo y la mantenibilidad del código.

Además, Angular utiliza HTML para las plantillas de los componentes y CSS/SCSS para los estilos. La combinación de TypeScript, HTML y CSS permite a los desarrolladores crear aplicaciones web ricas e interactivas con una estructura clara y modular.

TypeScript es fundamental en Angular, ya que permite definir clases, interfaces y tipos de datos, lo que facilita la detección temprana de errores y mejora la calidad del código. También permite el uso de características modernas de JavaScript, como decoradores y módulos, que son esenciales para la arquitectura basada en componentes de Angular.

d) ¿Qué prerequisitos necesita?

Para trabajar con Angular, es necesario tener conocimientos básicos de HTML, CSS y JavaScript. Además, es recomendable familiarizarse con TypeScript, ya que es el lenguaje principal utilizado en Angular.

También es útil tener experiencia con conceptos de desarrollo web, como el modelo de objetos del documento (DOM), solicitudes HTTP y programación asíncrona (promesas, async/await).

Además, es importante contar con un entorno de desarrollo adecuado, que incluya Node.js y npm (Node Package Manager) para gestionar las dependencias y utilizar Angular CLI, la herramienta oficial para crear y gestionar proyectos Angular.
