Actividad 2: Tutorial sobre Angular.

6. Ejemplo: Hola Mundo con Angular (CLI)

En Angular lo habitual es utilizar Angular CLI para crear y servir la aplicación. A continuación tienes los pasos y los fragmentos de código básicos que forman la aplicación mínima.

Crear y ejecutar el proyecto

npm install -g @angular/cli
ng new hola-angular
cd hola-angular
ng serve
# Abrir http://localhost:4200

Archivos principales (ejemplo)

Ejemplo de los ficheros que verás en src/app:

// src/app/app.component.ts
import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.css']
})
export class AppComponent {
  title = 'Hola desde Angular';
  count = 0;
  increment(){ this.count++; }
}

// src/app/app.component.html
<div>
  <h3>{{ title }}</h3>
  <p>Contador: {{ count }}</p>
  <button (click)="increment()">Aumentar</button>
</div>

// src/main.ts (arranque)
import { platformBrowserDynamic } from '@angular/platform-browser-dynamic';
import { AppModule } from './app/app.module';

platformBrowserDynamic().bootstrapModule(AppModule)
  .catch(err => console.error(err));
